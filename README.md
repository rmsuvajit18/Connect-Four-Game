# Connect Four Game

A classic Connect Four game built with HTML, CSS, and JavaScript.

## Overview

Connect Four is a two-player game where players take turns dropping colored discs into a 7-column, 6-row grid. The first player to connect four discs in a row (horizontally, vertically, or diagonally) wins.

## Project Structure

```
connect-four-game/
├── index.html      # Main HTML file
├── css/
│   └── style.css   # Styling
└── js/
    └── script.js   # Game logic
```

## How to Play

1. Open `index.html` in a web browser
2. Player 1 (Red) goes first
3. Click on any column to drop your disc into the lowest available row
4. Players alternate turns (Red → Yellow → Red...)
5. Connect four discs in a row to win
6. Click "Restart" to reset the game

## Features

- 7×6 game board
- Two-player local gameplay
- Visual turn indication (red/yellow discs)
- Restart button to reset the game

## Technologies

- **HTML5** - Page structure
- **CSS3** - Styling and grid layout
- **JavaScript** - Game logic and DOM manipulation
